## 🧠 Agent Instructions

When building this portfolio, agents must:

1. **Read all specs files before writing any code.**
2. **Never hardcode theme colors** — always use CSS custom properties (`--color-accent`, etc.).
3. **Implement all 3 profiles** as data attributes on `<html>` (e.g. `data-profile="dev"`).
4. **Dark/light mode** is a separate toggle from profiles — both must work in combination (6 total variants).
5. **Mobile-first** — all layouts start from 375px and scale up.
6. **Single HTML file** output — CSS and JS embedded.
7. **No external UI frameworks** except Tailwind utility classes if using React artifact.
8. **Sections order:** Hero → About → Skills → Experience → Projects → Testimonials → Contact.
9. **All placeholder content** is marked with `[PLACEHOLDER]` in `specs/04-content.md` — do not invent real data.
10. **Accessibility:** semantic HTML, aria-labels on interactive elements, keyboard-navigable theme switcher.
